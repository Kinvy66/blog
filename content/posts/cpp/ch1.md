第一章是C++的概览

## 1. 一个简单的C++程序

从一个简单的C++程序开始

```c++
int main()
{
    return 0;
}
```

由这个程序简单的介绍了函数的定义。

简单的介绍了c++程序在不同的平台编译运行的方式。



## 2. 输入和输出

c++的输入和输出 `cin` `cout` 以及对应的运算符 `<<` `>>`

```c++
std::cout << "hello" << std::endl;
std::cin >> val;
```



## 3. 注释

c++的注释两种：

```c++
//注释1
std::cout << "hello" << std::endl;
/*
注释2
*/
std::cin >> val;
```
